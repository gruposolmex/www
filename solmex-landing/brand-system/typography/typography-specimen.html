06 — Guidelines

Do's & Don'ts

Rules for maintaining typographic consistency.

✓ Do
Use Space Grotesk for headlines
Headlines should always use Space Grotesk for brand consistency.
✕ Don't
Use Inter for headlines
Inter should be reserved for body text and UI elements only.
✓ Do
USE WIDE TRACKING FOR LABELS
Small caps and labels should use increased letter-spacing.
✕ Don't
use tight tracking for labels
Tight tracking on small text reduces legibility.
✓ Do
Maintain comfortable line lengths between 50-75 characters for body text to ensure optimal readability.
Keep paragraphs at readable widths (50-75 characters).
✕ Don't
Extend text lines beyond comfortable reading widths which forces the eye to travel too far and makes it difficult for readers to find the start of the next line when they reach the end of the current one.
Avoid overly wide text columns.
