Brand Patterns

These patterns add visual interest while maintaining brand consistency. Use sparingly to enhance backgrounds and create depth.

45° Diagonal Lines
Subtle diagonal stripes at the brand's signature 45° angle.
Use: Section backgrounds, headers
Dot Grid
Evenly spaced dots on an 8px baseline grid.
Use: Presentation slides, technical documents
Grid Lines
Clean orthogonal grid suggesting precision.
Use: Technical specifications, data displays
Gradient Fade
Soft gradient wash adding warmth and depth.
Use: Hero sections, feature highlights
Topographic
Concentric circles suggesting movement and logistics.
Use: Dark backgrounds, premium materials
Chevron
Directional pattern indicating progress.
Use: Progress sections, call-to-action areas
Subtle Noise
Texture overlay adding depth to flat colors.
Use: Hero images, photography overlays
Corner Accent
Triangular corner element echoing the logo cut.
Use: Cards, documents, certificates

Implementation

/* Diagonal Lines Pattern */
.pattern-diagonal {
  background:
    repeating-linear-gradient(
      45deg,
      transparent,
      transparent 10px,
      rgba(255, 148, 59, 0.1) 10px,
      rgba(255, 148, 59, 0.1) 20px
    );
}

/* Dot Grid Pattern */
.pattern-dots {
  background-image:
    radial-gradient(circle, #FF943B 1.5px, transparent 1.5px);
  background-size: 24px 24px;
}

/* Grid Lines Pattern */
.pattern-grid {
  background-image:
    linear-gradient(rgba(255, 148, 59, 0.15) 1px, transparent 1px),
    linear-gradient(90deg, rgba(255, 148, 59, 0.15) 1px, transparent 1px);
  background-size: 24px 24px;
}
