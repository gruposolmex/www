05 — Usage Guidelines

Color Distribution

Maintain visual balance with the 60-30-10 rule.

60%
30%
10%
Neutral Tones 60%
Primary Orange Spectrum 30%
Accent & Semantic 10%

Application Rules

  • Use neutrals for backgrounds, text, and large surface areas
  • Reserve orange gradient for key UI elements and CTAs
  • Apply accent colors sparingly for status and feedback
  • Never use multiple accent colors in close proximity
  • Maintain high contrast for all text elements

08 — Implementation

CSS Variables

Copy these CSS custom properties to implement the Solmex color system.

/* Grupo Solmex - Color System */
:root {
  /* Primary Brand Spectrum */
  --solmex-orange: #FF943B;
  --solmex-orange-alt: #FF9C37;
  --solmex-amber: #FFAD5C;
  --solmex-gold: #FFC67D;
  --solmex-sunrise: #FFD91C;
  
  /* Gradients */
  --gradient-primary: linear-gradient(180deg, #FF9C37 0%, #FFD81C 100%);
  --gradient-45: linear-gradient(45deg, #FF943B 0%, #FFD91C 100%);
  --gradient-90: linear-gradient(90deg, #FF943B 0%, #FFD91C 100%);
  
  /* Industrial Neutrals */
  --carbon-black: #0A0A0A;
  --graphite: #1A1A1A;
  --steel-gray: #2C3E44;
  --iron: #4A5568;
  --concrete: #8B9197;
  --aluminum: #B0B5BA;
  --mist: #E4E7E9;
  --polar: #F8F9FA;
  
  /* Semantic Colors */
  --signal-red: #DC2626;
  --safety-green: #16A34A;
  --warning-amber: #F59E0B;
  --info-blue: #3B82F6;
}
