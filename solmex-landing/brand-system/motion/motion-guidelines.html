Motion & Animation

Guidelines for creating purposeful, delightful motion in Grupo Solmex interfaces

Motion Principles

Purposeful
Every animation serves a function—guiding attention, providing feedback, or enhancing understanding. Avoid decoration-only motion.
Quick & Responsive
Animations should feel snappy and responsive. Users shouldn't wait for animations to complete before continuing.
Natural
Motion should feel physical and natural. Use easing functions that mimic real-world physics.
Consistent
Similar elements should animate in similar ways. This builds familiarity and reduces cognitive load.

Logo Animation

Floating Logo

The Solmex logo may use a subtle floating animation to add life to static pages. Keep the movement gentle and elegant.

/* Logo Float Animation */
@keyframes logoFloat {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}

.logo { animation: logoFloat 6s ease-in-out infinite; }

Staggered Animations

Sequential Reveal

When multiple elements enter the view, stagger their animations with 100ms delays for a polished, orchestrated feel.

/* Stagger Pattern */
.item:nth-child(1) { animation-delay: 0.1s; }
.item:nth-child(2) { animation-delay: 0.2s; }
.item:nth-child(3) { animation-delay: 0.3s; }
/* Continue pattern... */
