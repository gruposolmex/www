# Grupo Solmex - Landing Page

Premium industrial logistics landing page for Grupo Solmex, built with Next.js 16, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Modern Tech Stack**: Next.js 16 (App Router), TypeScript, Tailwind CSS
- **Premium Animations**: Smooth parallax scrolling, section reveals with Framer Motion
- **Industrial Design**: Dark, professional aesthetic with orange/yellow accents
- **Fully Responsive**: Optimized for all devices
- **Performance Optimized**: Lazy loading, optimized animations

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in development.

### Production Build

```bash
npm run build
npm start
```

## Deployment

Ready for deployment to Vercel, Netlify, or any Node.js hosting platform.
