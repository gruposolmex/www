04 ‚Äî Installation

How to Install

Follow these steps to install the fonts on your operating system.

üìò Installation Steps

  1. Download the fonts: Click any of the download links above to visit the official source. Download the font files (usually .ttf or .otf format).
  2. Extract the files: If you downloaded a ZIP file, extract it to a folder on your computer.
  3. Install on your system:
    • Windows: Right-click each font file ‚Üí "Install" or drag to C:\Windows\Fonts
    • macOS: Double-click each font file ‚Üí "Install Font" or use Font Book app
    • Linux: Copy to ~/.fonts/ or ~/.local/share/fonts/
  4. Restart applications: Close and reopen any applications (Word, PowerPoint, etc.) to use the new fonts.
  5. Verify installation: Open a document editor and check if the fonts appear in your font list.
